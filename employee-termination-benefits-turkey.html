Severance, Notice & Unemployment Pay Guide for Employees in Türkiye

June 2026 5 min read Updated: July 12, 2026

Understanding your rights regarding severance pay, notice pay, and unemployment benefits in Türkiye can be complex. The entitlement to these benefits heavily depends on the specific reason for employment termination, denoted by the Social Security Institution (SGK) termination codes.

Below is a comprehensive table outlining whether an employee is entitled to Severance Pay (Kıdem Tazminatı), Unemployment Pay (İşsizlik Maaşı), and Notice Pay (İhbar Tazminatı) based on common termination codes and reasons.

Code Reason for Leaving Severance Pay Unemployment Pay Notice Pay
Code 3 Resignation (voluntary) Cannot Get Cannot Get Cannot Get
Code 3 Personal/Family reasons, relocation Cannot Get Cannot Get Cannot Get
Code 4 Unfair dismissal by employer Gets Gets Gets
Code 25 Dismissal upon necessity / Justified termination by employer (except malicious intent) Gets Gets Cannot Get
Code 24 Justified resignation by employee (mobbing, health, unpaid wages, etc.) Gets Gets Cannot Get
Code 12 Military service (Men) Gets Gets Gets
Code 13 Marriage (Women within 1 year) Gets Cannot Get Cannot Get
Code 13 Retirement eligibility: 15 years/3600 premium days (Before 08.09.1999) Gets Cannot Get Cannot Get
Code 14 Retirement eligibility: 25 years/4500-7000 premium days (Before 30.04.2008) Gets Cannot Get Cannot Get
Code 14 Retirement eligibility: 4600-5400 premium days (After 01.05.2008) Gets Cannot Get Cannot Get
Code 16 Transfer to another workplace of same employer Cannot Get Cannot Get Cannot Get
Code 17 Workplace closure Gets Gets Gets
Code 24 Health reasons (Resignation) Gets Gets Cannot Get
Disclaimer: The information provided in this table is for general informational purposes only and does not constitute legal advice. Employment disputes are highly factual, and variations may apply based on individual circumstances, employment contracts, and updated court precedents.
